# MESA Voice Recordings

This directory contains voice recordings for MESA's voice interface.

## Required Files

- `mesa_voice_chat_not_ready.wav` - Placeholder message for when voice chat is not yet available

## Adding the Voice File

Place your `mesa_voice_chat_not_ready.wav` file in this directory. The file should contain MESA's voice saying something like:

> "Voice interaction with MESA is not available yet. Please use the text chat or expert system interface for now. We're working hard to bring you voice capabilities soon."

## Technical Requirements

- Format: WAV (recommended for best browser compatibility)
- Sample Rate: 44.1kHz or 48kHz
- Channels: Mono or Stereo
- Duration: Keep it under 30 seconds for optimal user experience
