### **I. Microcirculatory Severity Logic (Layer 1: Pathophysiology)**
*   **Mechanical Sequestration Rule:** IF (Pathogen = *P. falciparum*) THEN (Mechanism = Cytoadherence to vascular endothelium) AND (Result = Sequestration of parasitized Red Blood Cells).
*   **Organ-System Mapping:** 
    *   IF (Sequestration = Brain) THEN (Inference = Cerebral Malaria/Sudden Convulsions).
    *   IF (Sequestration = Lung) THEN (Inference = ARDS/Respiratory Distress).
    *   IF (Sequestration = Placenta) THEN (Inference = Maternal/Neonatal Risk).

### **III. Pediatric Shock Resuscitation Logic (Layer 3: Nutrition Interaction)**
*   **Normal Nutritional Status Protocol:** IF (Diagnosis = Severe Malaria with Shock) AND (Nutrition = Normal) THEN **Action:** Administer **10–20 ml/kg** Ringer’s Lactate or Normal Saline over 30-60 minutes.
*   **Severe Acute Malnutrition (SAM) Protocol:** IF (Diagnosis = Severe Malaria with Shock) AND (Nutrition = SAM) THEN **Action:** Administer **10–15 ml/kg** over 1 hour using a **half-Ringer's + half-Dextrose 5%** solution.

### **IV. Advanced Differential: Rash and Ocular Pathognomony (Layer 4)**
*   **The "Rose Spot" Exclusion:** IF (Fever = High) AND (Rash = Sparse pink macules on trunk/Rose spots) THEN **Weight:** -20 for Malaria, +20 for Typhoid.

### **V. Expert System Recursive "Rule-Out" Logic (Layer 5)**
*   **The Triple-Set Rule:** IF (Clinical Suspicion = High) AND (Initial Smear = Negative) THEN **Requirement:** Repeat blood smears every **12–24 hours for a total of three sets** before ruling out malaria.
